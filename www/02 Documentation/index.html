B) CSS Files and Structure - top

This template imports following css files.

  1. "assets/css/bootstrap.min.css" - Is the bootstrap foundation css file.
  2. "assets/css/font-awesome.min.css" - Is icon pack css file.
  3. "assets/css/animate.css" - Is the animation css file.
  4. "assets/css/style.css" - Is the main css file.

C) JavaScript - top

This template imports following Javascript files.

  1. "assets/js/jquery-1.11.1.min.js" - jQuery JavaScript Library
  2. "assets/js/bootstrap.min.js" - Twitter Bootstrap framework
  3. "assets/js/jquery.cycle.min.js" - Jquery Cycle plugin
  4. "assets/js/jquery.backstretch.min.js" - Jquery Backstrech plugin
  5. "assets/js/jquery.mb.YTPlayer.js" - Jquery Youtube Player plugin
  6. "assets/js/script.js" - Main File
